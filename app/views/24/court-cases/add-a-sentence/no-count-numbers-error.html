There are missing count numbers

You are adding a consecutive offence, but the other offences do not have count numbers.

To add more than one consecutive offence, you must add the count numbers to the other offences.

How to add count numbers

On the Add offences page, for each offence:

  1. Select 'Edit'
  2. Enter the count number
  3. Select 'Accept changes'
  4. Once all the count numbers have been added, you can then select the consecutive offences
