Add offences

Let users add one or more offences in a structured, repeatable flow.

DWP Design System

This pattern is similar to the Add another thing pattern in the DWP Design System.

When to use it

Use this pattern when users need to add one or more offences to a court appearance.

It helps ensure offence details are entered in a consistent format and can be reviewed before continuing.

How it works

  • The user is taken through a structured journey to the first offence
  • They are then taken to the offence summary page
  • On the summary page, users can:
    • add another offence
    • edit an existing offence
    • delete an offence
    • continue to the next task
