Offence card Custom component

Use the offence card to display offence and sentence information in a clear, structured format.

When to use it

Use this pattern when users need to review or manage multiple offences or sentences in one place.

When not to use it

Do not use the offence card when the user needs to select an offence or sentence — for example, in a list of radio buttons.

How it works

Show missing information

Sometimes you might need to show where there is missing information on a sentence or offence card.

Show a link to the relevant question so the user can add the missing information. The link should take the user to the appropriate question page, and return them to the page with the offence card once the information has been entered.
