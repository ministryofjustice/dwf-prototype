Contextual question

Help users understand the context of a question that refers to previously entered information.

When to use it

Use this pattern in journeys with multiple pages, where the context of a question — such as referring to previously entered information — might be lost or forgotten by the user.

How it works

Show the question on the page as usual. If it relates to previously entered information and context may be needed to help the user understand the question, display the context below using the govuk-body-l class.

It can be helpful to use the word "this" in the question to refer to the context.

Add offences Select the sentence this is consecutive to
