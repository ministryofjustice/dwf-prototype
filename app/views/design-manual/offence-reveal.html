Offence reveal Custom component

Use the offence reveal to show or hide offences when they don’t need to be visible by default.

When to use it

Use this component on pages that include multiple court cases or appearances with offences.

Similar components

The offence reveal is similar to the details component in the GOV.UK Design System.

When not to use it

Do not use the offence reveal in the context of a single appearance, where key information should remain visible.

Only use this component to show offences or sentences. Offences that require user actions (such as edit or delete) should not be shown in an offence reveal.

How it works

The offence reveal is a toggle link that shows or hides a list of offences or sentences when clicked.

The text changes between ‘Show offences’ and ‘Hide offences’ depending on the state.

Each offence should be displayed using the offence card component.
