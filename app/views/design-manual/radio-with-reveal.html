Radio with reveal GOV.UK Design System

Use the radio with reveal to capture additional information when a user answers a question.

When to use it

If a user selects an option that requires additional information, use a radio with reveal to capture it. This avoids introducing additional pages and keeps journeys shorter.

GOV.UK Design System

Radio with reveal is part of the GOV.UK Design System radios component.

When not to use it

Do not use a radio with reveal when the additional information required has more than one part. Use multiple stages instead.

How it works

When the user answers a question that requires additional information, reveal the follow-up question when they select the relevant option.

Make sure that any revealed fields are only submitted when visible and completed. For example, if a user enters information in a reveal but then selects a different option, do not submit the information in the reveal. This prevents validation issues.
