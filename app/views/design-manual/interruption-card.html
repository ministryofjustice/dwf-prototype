Interruption card

Highlights important information that users need to see before continuing.

When to use it

  • To warn users about possible errors
  • When the source of an error cannot be shown next to the input field
  • To provide users with clear instructions on how to resolve an issue

How it works

  • Displays an interruption card containing key information
  • Provides guidance on how to resolve the error when the source isn’t visible
  • The call to action (CTA) depends on the type of interruption:
  • Possible error: Users can either acknowledge and continue or return to the previous page to check their input
  • Instructions to fix an error: Users are directed to the page where they can resolve the issue or go back a step if needed

Research & rationale

  • Ensures users are made aware of important errors
  • Helps reduce confusion by explaining how to fix issues when direct inline error messaging isn’t possible
