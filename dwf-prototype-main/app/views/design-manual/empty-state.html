Empty state

Use an empty state to show users when there are no records to display and guide them towards the next action.

When to use it

Use an empty state when a user visits a page that will eventually display a list of items — such as offences or court cases — but no data exists yet.

Empty states help:

  • set user expectations
  • maintain a consistent page layout
  • provide a clear next step, such as a link to add the missing item

Use them in journeys where users may return to a page later, so they are not confused by sudden changes or unsure where items might appear.

How it works

Empty state with call to action

Use standard body text to let the user know no items have been added yet. Include a link or button to add the relevant item.

Court cases

There are no court cases recorded for Daryl Evans.

Add a court case now

Contextual empty state

Use inset text when items might later appear in a section of the page.

This helps the user understand where information may appear, and prevents confusion when sections appear empty.

There are no offences with non-custodial outcomes.
