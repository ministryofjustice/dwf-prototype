Appearance information panel Custom component

Use this component when the user is in the context of a specific court appearance.

When to use it

Use the appearance information panel when the user is on a journey within the context of a single appearance.

When not to use it

Do not use the appearance information panel if that information is already shown on the page, or if the page includes multiple court cases or appearances.

How it works

Place the panel in a one-quarter-width column using the class govuk-grid-column-one-quarter.
