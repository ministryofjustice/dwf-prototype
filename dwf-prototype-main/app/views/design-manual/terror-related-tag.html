Terror related tag

Visually indicates when an offence has a terrorism-related element.

When to use it

  • Any offence identified as having a terrorism-related element
  • Where users need to be aware that this might affect release date calculations

How it works

  • Uses the GOV.UK tag component to highlight terrorism-related offences

Research & rationale

  • Users have responded well to the tag in user research
  • Provides a clear, non-intrusive visual cue aligned with GOV.UK conventions
