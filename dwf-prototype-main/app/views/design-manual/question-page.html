Question page

We follow theÂ GOV.UK Question page pattern, using a one-thing-per-page approach.

When to use it

  • When asking users a single, clear question on a page
  • To keep journeys simple and focused

How it works

  • Uses GOV.UK large heading (govuk-heading-l) for the question heading, which is more suitable for longer journeys
  • The question is always the page title
  • We use govuk-caption to show the task the user is on
